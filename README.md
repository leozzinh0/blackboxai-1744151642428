# ClickNote

ClickNote é sua plataforma de notas online favorita, projetada para ajudá-lo a organizar suas ideias, projetos e tarefas de forma simples e eficiente. Esta aplicação é ideal para quem busca praticidade e acessibilidade em um só lugar.

## Project Overview

ClickNote oferece recursos incríveis para criar e gerenciar notas, com a possibilidade de sincronização entre dispositivos, assegurando a segurança de suas informações com criptografia de ponta a ponta. Experimente gratuitamente por 14 dias e veja como melhorar sua produtividade!

## Installation

Para rodar o ClickNote localmente, você pode simplesmente abrir o arquivo `index.html` em um navegador da sua escolha. Não há necessidade de instalação adicional, pois todos os recursos necessários já estão integrados!

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.

## Usage

Após abrir o arquivo `index.html`, você poderá visualizar a interface do ClickNote. Navegue pelas seções para explorar suas funcionalidades, como:

- Criar e organizar notas
- Acompanhar suas tarefas
- Acessar notas em diferentes dispositivos

## Features

- **Notas Rápidas:** Crie e organize suas notas de forma simples e intuitiva.
- **Sincronização:** Acesse suas notas de qualquer dispositivo, a qualquer momento.
- **Segurança:** Suas informações são protegidas com criptografia de ponta a ponta.
  
## Dependencies

O ClickNote utiliza as seguintes bibliotecas:

- [Tailwind CSS](https://tailwindcss.com/) - Framework CSS para design responsivo
- [Font Awesome](https://fontawesome.com/) - Ícones sociais e de interface
- Google Fonts - Fonte Poppins para tipografia

As referências estão embutidas no arquivo HTML, portanto não é necessário instalar dependências externas.
